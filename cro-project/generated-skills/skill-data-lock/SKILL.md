## Constraints / 约束

### Must (必须)
- **C001**: **必须**完成全部受试者的计划访视。
- **C002**: **必须**完成全部安全性和疗效数据的收集。
- **C004**: **必须**完成 100% 的源数据核查（SDV）。
- **C005**: **必须**关闭所有待处理的质疑（开放质疑数量为零）。
- **C006**: **必须**完成所有方案偏离的评估和关闭。
- **C007**: **必须**由医学监查员（MA）完成主要疗效数据的医学审核。
- **C008**: **必须**由数据经理完成数据质量审核报告。
- **C009**: **如果** 存在尚未解决的医学问题，**必须**在锁定前完成医学裁决，并形成书面记录。
- **C012**: **必须**由数据经理确认全部检查项目通过后方可发起锁定流程。
- **C014**: **如果** 解锁涉及受试者知情同意相关问题，**必须**先获得伦理委员会的书面意见。
- **C017**: 解锁后**必须**重新执行相关核查流程，确认修正正确性。
- **C018**: **必须**填写《数据解锁记录表》，记录解锁原因、批准人、执行人及解锁范围。

### Should (应当)
- **C003**: **如果** 存在受试者提前退出或失访，**应当**记录原因并在锁定报告中说明。
- **C010**: **如果** 存在影响主要疗效终点的不良事件，**应当**由医学监查员确认评估无误后方可锁定。
- **C011**: **如果** 申办方要求在锁定前进行统计专家审核，**应当**在满足统计审核要求后执行锁定。
- **C013**: **应当**将检查清单作为锁定报告附件存档。
- **C015**: **如果** 解锁需要修改统计计划，**应当**由统计师确认后执行。
- **C016**: **如果** 解锁涉及跨中心数据，**应当**评估对整体数据的影响。
- **C019**: 解锁记录**应当**作为项目档案永久保存。
